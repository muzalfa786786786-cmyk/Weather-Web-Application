# Weather-Web-Application
 I have successfully developed a fully functional Professional Multi-Page Weather Web Application using HTML, CSS, JavaScript, and OpenWeatherMap API.
During this project, I worked on:
```
 ✅ Login Page with validation
 ✅ Weather Search functionality (City-based search)
 ✅ API Integration using Fetch()
 ✅ Dynamic background based on weather conditions
 ☀️ Sunny → Bright theme
 🌧 Rainy → Rain animation
 ☁️ Cloudy → Cloud theme
 ✅ DOM Manipulation & Event Handling
 ✅ Responsive and modern UI design
 ✅ Glassmorphism UI styling
```
